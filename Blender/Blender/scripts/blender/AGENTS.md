# scripts/blender/AGENTS.md — Blender Technical Artist Rules

- Scripts must run with Blender Python.
- Support SVG line-art export, GLB export, and PNG preview/turntable.
- Use clear CLI arguments.
- Do not assume Blender is installed.
- Return readable error messages.
- Never make Blender mandatory for IDE runtime.
- Prefer named collections: Avatar, Rig, Export, Guides, Ignore.
- Use orthographic camera for line art.
